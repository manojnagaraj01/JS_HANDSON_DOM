Question:08 What's the difference between window vs document?

In the context of web development and browser scripting, "document" and "window" refer to two different objects that are part of the Document Object Model (DOM) and represent different aspects of a web page. Here's the difference between them:

Window Document
The window object represents the browser window or tab in which the web page is loaded. The document object represents the HTML content of the current web page. It is a property of the global window object
It serves as the global object in the browser's JavaScript environment and provides methods and properties for controlling the browser and interacting with the user. It provides an interface to access and manipulate the elements, content, and structure of the web page.
The window object includes methods for opening new browser windows or tabs (open()), navigating to different URLs (location), setting timeouts/intervals (setTimeout(), setInterval()), and handling user interactions. You can use methods like getElementById(), querySelector(), and properties like innerHTML to modify the content of the web page.
The window object also encompasses properties related to the browser environment, such as the screen dimensions, history, and more. The document object is essentially a representation of the entire HTML document, including all its elements, text, and attributes.

In summary, the document object is focused on the content and structure of the web page itself, while the window object provides control over the browser window and its interactions with the user. They are both integral parts of web development and are used to create dynamic and interactive web pages.
